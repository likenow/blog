## git 推送大文件buffer设置

1. 打开`SourceTree`,点击右上角的设置

2. 在弹出的框中,点击左下角的编辑配置文件

3. 在弹出的文本文件（Git 配置文件）中最下行加入如下代码:

```
[http]
    postBuffer = 524288000
```
