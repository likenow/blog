### 用 c++ 编写桌面程序的最佳方法 Best way to make desktop applications in c++

Dear ImGui

imGUI 如果你有非常大的数据集，如果他们在每一帧都被渲染刷新，你可能会遇到一些问题。

即时模式 immediate mode

每次都会重新绘制所有的元素，绝大多数游戏是这类模式。

保留模式 retain mode

当需要改变时，执行的是改变的处理，绝大多数应用程序是用的这种模式。
